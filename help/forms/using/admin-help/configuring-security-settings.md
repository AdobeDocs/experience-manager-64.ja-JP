# セキュリティ設定の指定{#configuring-security-settings}

パスワードを設定したり、印刷や編集など特定の機能を制限したりして、PDF ドキュメントへのアクセスを制限できます。PDF ドキュメントに制限された機能が含まれている場合、それらの機能に関連するツールやメニュー項目は淡色表示されます。また、ドキュメントの暗号化や認証など、他の方法を使用して保護されたドキュメントを作成することもできます。セキュリティ設定には、特定の PDF 変換に使用するパスワードと固有のオプションがあります。

セキュリティ設定ページでは、次のタスクを実行できます。

## セキュリティ設定の作成または編集 {#create-or-edit-a-security-setting}

*セキュリティ設定*&#x200B;によって、そのセキュリティ設定を使用して変換されたファイルのセキュリティと権限が制御されます。

1. 管理コンソールで、サービス／PDF Generator／セキュリティ設定をクリックします。
1. 「新規作成」をクリックするか、セキュリティ設定の名前をクリックします。
1. セキュリティ設定の新規作成／編集ページで、セキュリティ設定の必須情報を入力します（[ファイルタイプ設定の指定](/help/forms/using/admin-help/configuring-file-type-settings.md#configuring-file-type-settings)を参照）。
1. 「保存」をクリックし、表示されたダイアログボックスに設定の名前を入力して、「OK」をクリックします。

### セキュリティ設定 {#security-settings}

次の設定で、互換性と暗号化を設定します。セキュリティ設定へのアクセス方法については、[セキュリティ設定の作成または編集](configuring-security-settings.md#create-or-edit-a-security-setting)を参照してください。

**互換性：** パスワードで保護されたドキュメントを開くための暗号化の種類を設定します。 「Acrobat 3.0 以降」オプションでは、低い暗号化レベルを使用し、その他のオプションでは高い暗号化レベルを使用します。

**Acrobat 3.0 以降：** 低い暗号化（40 ビット RC4）を使用します。

**Acrobat 5.0 以降：** 高い暗号化（128 ビット RC4）を使用します。

**Acrobat 6.0 以降：** 高い暗号化（128 ビット RC4）を使用します。 このオプションでは、メタデータで検索を有効にできます。

**Acrobat 7.0 以降：** 高い暗号化（128 ビット AES）を使用します。 このオプションでは、メタデータでの検索と、添付ファイルのみの暗号化を有効にできます。

**Acrobat 9.0 以降：** 高い暗号化（256 ビット AES）を使用します。 このオプションでは、メタデータでの検索と、添付ファイルのみの暗号化を有効にできます。

古いバージョンの Acrobat では、そのバージョンよりも上位の互換性が設定されている PDF ドキュメントを開くことはできません。例えば、「Acrobat 7.0 以降」オプションを選択した場合は、そのドキュメントを Acrobat 6.0 およびそれより前のバージョンで開くことはできません。

互換性レベルが同じソースの PDF の互換性と一致することを確認します。例えば、標準の PDF 設定（Acrobat 5.0 以降との互換性があります）を使用するように設定された監視フォルダーがある場合は、セキュリティの互換性レベルを Acrobat 5.0 より上位にすることはできません。

**ドキュメントの制限：** 使用可能なドキュメントの制限は、選択した「互換性」オプションによって異なります。

**暗号化なし：** ドキュメントのどの部分も暗号化しません。

**すべてのドキュメントコンテンツを暗号化：** ドキュメントとドキュメントのメタデータを暗号化します。 このオプションを選択した場合、検索エンジンはドキュメントのメタデータにアクセスできません。

**メタデータを除くすべてのドキュメントコンテンツを暗号化する (Acrobat 6以降互換 ):** ドキュメントのコンテンツを暗号化しますが、検索エンジンはドキュメントのメタデータにアクセスできます。 このオプションは、「互換性」オプションが Acrobat 6.0 以降、Acrobat 7.0 以降または Acrobat9.0 以降に設定されている場合にのみ使用できます。

**添付ファイルのみを暗号化する (Acrobat 7以降と互換 ):** ユーザーはパスワードなしでドキュメントを開くことができますが、添付ファイルを開くにはパスワードを入力する必要があります。 このオプションは、「互換性」オプションが Acrobat 7.0 以降または Acrobat 9.0 以降に設定されている場合にのみ使用できます。

次の設定で、パスワードのセキュリティを設定します。

>[!NOTE]
>
>パスワードを忘れた場合は、ドキュメントから復元することはできません。忘れた場合のために、パスワードは別の安全な場所に保存しておくことをお勧めします。または、パスワードで保護されていないドキュメントのバックアップコピーをとっておきます。

**ドキュメントを開く際にパスワードが必要：** パスワードオプションを有効にします。

**ドキュメントを開くパスワード：** ユーザーが指定したパスワードを入力しない限り、ドキュメントを開かないようにします。 パスワードは、大文字と小文字が区別されます。Acrobat では、RSA Security Inc. の RC4 セキュリティ方式を使用して PDF ドキュメントをパスワードで保護します。印刷や編集を制限している場合は、セキュリティを強化するためにドキュメントを開くパスワードの追加をお勧めします。

**ドキュメントを開くパスワードを再入力：** ドキュメントを開くパスワードが正しいことを確認します。

**添付ファイルを開くためのパスワードを要求：** パスワードオプションを有効にします。 このオプションが使用可能になるのは、「互換性」オプションが Acrobat 7.0 以降または Acrobat 9.0 以降に設定されており、「ドキュメントに関する制限」オプションが「添付ファイルのみを暗号化する」に設定されている場合のみです。

**添付ファイルを開くパスワード：** 添付ファイルを開く際にパスワードが必要であることを確認します。 ユーザーはパスワードを使用しなくてもドキュメントを開くことができます。このオプションが使用可能になるのは、「互換性」オプションが Acrobat 7.0 以降または Acrobat 9.0 以降に設定されており、「ドキュメントに関する制限」オプションが「添付ファイルのみを暗号化する」に設定されている場合のみです。

**添付ファイルの再入力：** パスワードが正しいことを確認します。 このオプションが使用可能になるのは、「互換性」オプションが Acrobat 7.0 以降または Acrobat 9.0 以降に設定されており、「ドキュメントに関する制限」オプションが「添付ファイルのみを暗号化する」に設定されている場合のみです。

次のオプションで、権限を設定します。

**パスワードを使用して、ドキュメントの印刷および編集とセキュリティ設定を制限します。** 権限の制限を有効にします。

**権限パスワード：** ユーザーが印刷および編集できないように制限します。 ユーザーは、指定したパスワードを入力しない限りこれらのセキュリティ設定を変更することはできません。「ドキュメントを開くパスワード」で使用しているパスワードは使用できません。権限パスワードを設定すると、そのパスワードを入力したユーザーだけがセキュリティ設定を変更できます。PDF ドキュメントに両方の種類のパスワードが含まれている場合、いずれかのパスワードでドキュメントが開きますが、制限されている機能は権限パスワードでのみ設定または変更できます。PDF ドキュメントに権限パスワードのみが含まれている場合や、ユーザーがドキュメントを開くパスワードを使用してドキュメントを開いている場合は、ユーザーがセキュリティ設定を変更しようとすると、パスワードの入力を求めるプロンプトが表示されます。

**権限パスワードの再入力：** 権限パスワードが正しいことを確認します。

**印刷を許可：** 印刷ドキュメントの印刷品質をPDFします。

**なし：** ユーザーがドキュメントを印刷できないようにします。

**低解像度 (150 dpi):** ユーザーは 150 dpi 以下の解像度でドキュメントを印刷できます。 各ページはビットマップ画像として印刷されるため、印刷速度が低下することがあります。 このオプションは、高い暗号化レベル（Acrobat 5.0、6.0、7.0 または 9.0）が選択されている場合にのみ使用できます。

**高解像度：** ユーザーは任意の解像度で印刷でき、高品質のベクトル出力を、高度な高品質印刷機能をサポートする PostScript やその他のプリンターに向けることができます。

**変更が許可されました：** PDFドキュメントで許可する編集アクションを定義します。

**なし：** ユーザーが署名やフォームフィールドへの入力を含め、ドキュメントを変更できないようにします。

**ページの挿入、削除、回転：** ユーザーはページの挿入、削除、回転をおこなえるほか、しおりやサムネールページを作成できます。 このオプションは、高い暗号化レベル（Acrobat 5.0、6.0、7.0 または 9.0）が選択されている場合にのみ使用できます。

**フォームフィールドへの入力と既存の署名フィールドへの署名：** ユーザーがフォームに入力し、電子署名を追加できるようにします。 ただし、コメントを追加したり、フォームフィールドを作成したりすることはできません。このオプションは、高い暗号化レベル（Acrobat 5.0、6.0、7.0 または 9.0）が選択されている場合にのみ使用できます。

**コメント作成、フォームフィールドへの入力、既存の署名フィールドへの署名：** ユーザーがフォームに入力し、電子署名とコメントを追加できるようにします。

**ページレイアウト、タッチアップ、フォームフィールドへの入力、既存の署名フィールドへの署名：** ユーザーはページの挿入、回転、削除、しおりやサムネール画像の作成、フォームの入力、電子署名の追加を行うことができます。 このオプションでは、ユーザーはフォームフィールドを作成できません。このオプションは、低い暗号化レベル（Acrobat 3.0）が選択されている場合にのみ使用できます。

**ページの抽出を除くすべての操作：** ユーザーは、ページを削除する以外の、変更許可リストの任意の方法を使用してドキュメントを変更できます。

**テキスト、画像およびその他のコンテンツのコピーを有効にする：** ユーザーは、PDFドキュメントの内容を選択してコピーできます。 また、PDF ファイルのコンテンツへのアクセスを必要とするユーティリティ（Acrobat Catalog など）で、それらのコンテンツを取得することもできます。このオプションは、高い暗号化レベルが選択されている場合にのみ使用できます。

**視覚障碍者向けの画面Readerデバイスのテキストアクセスを有効にする：** 視覚障碍を持つユーザーがスクリーンリーダーを使用してドキュメントを読み取ることができます。 ただし、ユーザーはドキュメントのコンテンツをコピーしたり抽出したりすることはできません。このオプションは、高い暗号化レベルが選択されている場合にのみ使用できます。

## セキュリティ設定の削除 {#delete-a-security-setting}

不要になったセキュリティ設定は削除できます。ただし、事前に設定されているセキュリティ設定は削除できません。

1. 削除する設定の隣にあるチェックボックスを選択します。複数の設定を選択できます。
