## AEM 6.4.8.0 に含まれる機能 {#what-s-included-in-aem}

AEM 6.4.8.0は、新機能、お客様から要望された主な機能強化、パフォーマンス、安定性、セキュリティの向上を含む重要なアップデートです。2018年4月のAEM 6.4の一般リリース(GA)以降にリリ **ースされました。**

また、6.4.8.0には、その前にリリースされたすべてのAEM 6.4サービスパックが含まれています。

このサービスパックリリースの主なハイライトは次のとおりです。

* 組み込み型のリポジトリ（Apache Jackrabbit Oak）をバージョン 1.8.20 に更新しました。

* ワードラップ機能は、WCM-RTEの日本語のWebサイトでサポートされています。

* 日本語のWebサイトでは、改行と改行の処理がサポートされています。

* 日本語の文と行を適切な位置で区切るための、BUNSETメソッドの使用のサポートを追加しました。

* Correspondence ManagementのCCR UIで、ドイツ語ロケールの10進数値がサポートされるようになりました。

* SOAP Webサービスを使用したフォームデータモデルの統合で、要素の選択グループまたは属性がサポートされるようになりました。

* AEM Assetsは、Adobe I/Oを通じてBrand Portalで設定されるようになりました。

* ContextHubにバンドルされているjQueryバージョンを3.4.1に更新しました。

また、これは累積的な機能であり、6.4.7.0には、AEM 6.4のすべてのサービスパックのリリースが含まれています。

#### AEM 6.4.4.0 {#experience-manager-6440}

AEM 6.4.4.0 is an important update that includes performance, stability, security and key customer fixes and enhancements released since the general availability of AEM 6.4 in **April 2018.**

また、これは累積的な機能であり、6.4.4.0には、その前にリリースされたすべてのAEM 6.4サービスパックが含まれています。

AEM 6.4.4.0の主な特徴は次のとおりです。

* 組み込み型のリポジトリ（Apache Jackrabbit Oak）をバージョン 1.8.11 に更新しました。
* サービスバージョンHTTP要求が頻繁に発生するのを防ぐために、サービスバージョンのキャッシュのサポートを追加しました。
* 自動タグの削除のサポートを追加しました。
* カタログウィザードの無限スクロールを実装。
* コミュニティサイトに従って権限を制限するバックポート機能。
* Sling Granite Content Health Checkのパフォーマンスの修正(キャッシュ、非同期実行、除外リスト)。
* assetpickerボタンをpagethumbnailダイアログに追加しました。
* フィールド上でのツールチップの位置を許可する新しいプロパティを追加しました。
* マルチフィールド内のColorPickerのサポートを改善。
* コンテンツフラグメントのclientlibsにある数値入力マルチフィールドの空の値を無視するチェックを追加しました。
* Microsoft Translator Text API v3のサポートを有効にしました。

#### AEM 6.4.3.0 {#experience-manager-6430}

AEM 6.4.3.0は、パフォーマンス、安定性、セキュリティ、お客様向けの主要な修正、および2018年4月のAEM 6.4の一般リリース(GA)以降にリリースされた機能強化を含む重要なアップデートです。

また、これは累積的な機能であり、6.4.3.0には、AEM 6.4のすべてのサービスパックのリリースが含まれています。

AEM 6.4.3.0の主な特徴は次のとおりです。

* 組み込み型のリポジトリ（Apache Jackrabbit Oak）をバージョン 1.8.9 に更新しました。
* アダプティブフォームのテンプレートでallowedPathsプロパティのサポートを追加しました。
* AEMでのアセットのパネルベースの検索の強化
* ログインページのクロスサイトスクリプティング(XSS)に関する修正。
* UI計測が改善されました。
* FormDataの処理の改善。
* 複数フィールド内での項目の命名処理を改善。
* 選択中のプレースホルダー項目(カードの表示とリストの表示)の処理を改善。
* マネージドサービスのAdobe IMS認証と管理コンソールのサポートを追加しました。

AEM 6.4.2.0 is an important update that includes performance, stability, security and key customer fixes and enhancements released since the general availability of AEM 6.4 in **April 2018.**

* /etc/clientlibs/social/thirdparty/handlebars/source/handlebars.jsバージョン1.3に更新します。
* DMHybridオプトアウトおよびオプトインパッケージの修正点です。
* スマート切り抜きの修正点です。
* OOTB構成値を新しい場所（/etcから/conf）に移行しました。
* カラープロファイルを顧客設定に追加しました。
* /etc —&amp;gt；からのImage Server設定の移行/conf.
* 翻訳用のソースコンテンツフラグメントを追加しました。

* 電子メールの検証ARIAのサポートを追加しました。

* メタデータの読み込み/書き出し時に、リソースが見つかりませんというエラーが表示される。 CQ-4253263
* 6.4.2.0の上に6.4.3.0を適用した後で、イメージコンポーネントとページプロパティを編集できない。CQ-4260316およびCQ-4260441この問題を回避するには：
   * パッケージマネージャーに移動
   * パッケージ&quot;cq-ui-wcm-admin-content-1.0.1004.zip&quot;を再インストールします。
   * すべてのJSPを再コンパイ `(http://<AEM HOST>:<AEM PORT/system/console/slingjsp)` ルするか、インスタンスを再起動します。
